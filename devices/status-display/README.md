# Status Display

> **Status:** Planned

## Overview

ESP32-2432S028 (CYD / Cheap Yellow Display) module acting as a wall-mounted status display and control node.

## Hardware

- **Board:** ESP32-2432S028 (2.8" TFT + touch, ESP32-WROOM-32)
- **Display:** ILI9341 320x240 TFT
- **Touch:** XPT2046 resistive
- **Framework:** ESPHome or LVGL via ESPHome

## Planned Features

- Room temperature/humidity display (from BLE thermometers)
- HVAC mode & target temp control (Della mini splits, Ecobee)
- Irrigation status overview
- Quick-action touch buttons
